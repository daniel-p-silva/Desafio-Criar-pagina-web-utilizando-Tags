Objetivo:

O objetivo deste desafio é criar uma simples página utilizando as diversas tags HTML aprendidas durante as aulas.

Introdução:

HTML é uma linguagem de marcação utilizada para criar e estruturar páginas da web. Com o uso de diferentes tags, é possível formatar e organizar o conteúdo de maneira eficiente.

Tags Utilizadas:

  • h1 até h6: São utilizadas para definir diferentes níveis de títulos, do mais importante ao menos importante.
  • p: Utilizada para criar parágrafos no texto.
  • mark: Destaca o texto dentro do conteúdo.
  • small: Reduz o tamanho do texto para uma fonte menor.
  • i: Utilizado para colocar o texto em itálico.
  • u: Sublinha o texto.
  • strong: Destaca o texto de forma mais enfática.
  • ol e ul: Criam listas ordenadas e não ordenadas, respectivamente.
  • li: Utilizada dentro de ol ou ul para definir os itens da lista.
  • a: Cria um link para outras páginas ou recursos.
  • hr: Insere uma linha horizontal para separar o conteúdo.
  • sub e sup: Utilizadas para texto subscrito e sobrescrito, respectivamente.
  • font: Define o tamanho, a cor e o tipo de fonte do texto (essa tag não é mais recomendada e é melhor substituída por CSS).
  • blockquote: Utilizada para citações em bloco.
  • del: Indica texto que foi deletado ou removido.
  • abbr: Define uma abreviação ou acrônimo.

Conclusão:

Utilizando essas tags de forma adequada, é possível criar textos estruturados e visualmente atraentes em HTML.
